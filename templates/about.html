So what is this whole idea about ?

- This is Machine Learning application that tries to predict the next word as you type in a sentence.

Wait! How does that happen ?

- Well, It uses Neural Networks to learn from huge datasets and tries to predict the next word based on the previous words you have typed in.

- A Neural Network interprets sensory data, translating images, sound, text, and more into numerical patterns. It identifies and analyzes patterns for tasks like labeling and clustering.
